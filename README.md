# borancakir.com

Personal landing page for [borancakir.com](https://borancakir.com).

A single static HTML file served via GitHub Pages. No framework, no build step.

## Files

- `index.html` — the page (HTML, CSS, inline SVG icons)
- `CNAME` — custom domain configuration for GitHub Pages

## Local preview

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Deploy

Push to `main`. GitHub Pages rebuilds automatically.

## Fonts

- Geist — heading
- Newsreader — body

Both loaded from Google Fonts.
